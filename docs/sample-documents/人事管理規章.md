# 人事管理規章

## 第一章 總則

### 1.1 制定目的

為建立完善之人事管理制度，保障員工權益，提升組織效能，特訂定本規章。

### 1.2 適用對象

本規章適用於公司所有正式員工、約聘員工及實習生。

## 第二章 聘僱管理

### 2.1 聘僱原則

公司聘僱員工以公開、公平、公正為原則，不因種族、性別、年齡、宗教、政治立場等因素而有差別待遇。

### 2.2 試用期規定

- 新進員工試用期為三個月
- 試用期間表現優異者，得提前結束試用
- 試用期滿經考核合格者，正式任用

### 2.3 勞動契約

- 到職日起簽訂勞動契約
- 契約內容包含：職務、薪資、工時、福利等
- 雙方各執一份

## 第三章 差勤管理

### 3.1 工作時間

- 標準工時：每日 8 小時，每週 40 小時
- 上班時間：09:00 - 18:00（含午休 1 小時）
- 彈性上下班：可於 08:00-10:00 間打卡上班

### 3.2 出勤紀錄

- 員工應確實刷卡或使用差勤系統打卡
- 忘記打卡應於當日補登並經主管核准
- 出勤紀錄作為薪資計算依據

### 3.3 加班規定

#### 3.3.1 加班申請

- 加班需事先申請並經主管核准
- 申請應載明加班原因及預計時數
- 特殊狀況得事後補申請（限次日內）

#### 3.3.2 加班費計算

| 加班類型                | 倍率 |
| ----------------------- | ---- |
| 平日延長工時（前2小時） | 1.34 |
| 平日延長工時（後2小時） | 1.67 |
| 休息日（前2小時）       | 1.34 |
| 休息日（2-8小時）       | 1.67 |
| 國定假日                | 2.0  |

### 3.4 請假規定

#### 3.4.1 假別與天數

| 假別     | 天數    | 備註              |
| -------- | ------- | ----------------- |
| 特別休假 | 依年資  | 到職滿半年 3 天起 |
| 事假     | 14天/年 | 不給薪            |
| 病假     | 30天/年 | 半薪              |
| 婚假     | 8天     | 給薪              |
| 喪假     | 3-8天   | 依親等            |
| 產假     | 8週     | 給薪              |
| 陪產假   | 7天     | 給薪              |
| 公假     | 依需要  | 給薪              |

#### 3.4.2 請假流程

1. 登入人事系統提出申請
2. 檢附相關證明文件（如需）
3. 主管審核
4. 人資單位備查

## 第四章 薪酬福利

### 4.1 薪資結構

- 本薪：依職級及專業能力核定
- 職務加給：依職務性質給予
- 績效獎金：依個人及部門績效發放

### 4.2 薪資發放

- 發薪日：每月 5 日
- 遇假日提前至前一工作日
- 採銀行轉帳方式

### 4.3 福利項目

- 勞健保：依法投保
- 團體保險：公司全額負擔
- 員工旅遊：每年一次
- 三節禮金：端午、中秋、年終
- 生日禮金：當月發放
- 進修補助：依公司規定申請

## 第五章 績效考核

### 5.1 考核週期

- 年度考核：每年 12 月
- 期中面談：每年 6 月

### 5.2 考核項目

1. **工作績效**（50%）：目標達成率
2. **工作態度**（20%）：出勤、配合度
3. **專業能力**（20%）：技能提升
4. **團隊合作**（10%）：溝通協調

### 5.3 考核等級

| 等級 | 說明   | 建議調薪幅度 |
| ---- | ------ | ------------ |
| A    | 卓越   | 5-8%         |
| B    | 優良   | 3-5%         |
| C    | 合格   | 1-3%         |
| D    | 待改進 | 0%           |
| E    | 不合格 | 列入輔導     |

## 第六章 訓練發展

### 6.1 訓練類型

- **新人訓練**：到職後一週內完成
- **在職訓練**：持續性專業訓練
- **管理訓練**：主管階層必修
- **外部訓練**：依需求申請

### 6.2 訓練時數

- 每人每年最低訓練時數：20 小時
- 主管階層：30 小時

## 第七章 離職管理

### 7.1 離職預告

| 年資      | 預告天數 |
| --------- | -------- |
| 未滿3個月 | 無需預告 |
| 3個月-1年 | 10天     |
| 1-3年     | 20天     |
| 3年以上   | 30天     |

### 7.2 離職流程

1. 提出離職申請
2. 主管面談挽留
3. 辦理工作交接
4. 資產歸還（識別證、設備等）
5. 人資辦理離職手續
6. 核發離職證明
