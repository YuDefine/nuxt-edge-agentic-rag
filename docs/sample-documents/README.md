# 範例文件

此目錄包含用於測試文件上傳功能的範例檔案。

## 用途

- 開發時測試 `/admin/documents/upload` 上傳流程
- 驗證文件解析、分塊、向量化功能
- Demo 展示用

## 支援格式

| 格式     | 副檔名 | MIME Type         |
| -------- | ------ | ----------------- |
| 純文字   | `.txt` | `text/plain`      |
| Markdown | `.md`  | `text/markdown`   |
| PDF      | `.pdf` | `application/pdf` |

## 限制

- 單檔最大 10 MB
- 檔案不可為空

## 檔案清單

| 檔案                   | 類型     | 說明                              |
| ---------------------- | -------- | --------------------------------- |
| `採購流程操作手冊.md`  | Markdown | 完整的 SOP 文件，含表格、章節結構 |
| `人事管理規章.md`      | Markdown | 規章制度文件，含多層級標題        |
| `系統操作常見問題.txt` | 純文字   | FAQ 格式，適合測試 Q&A 擷取       |

## 使用方式

1. 進入 `/admin/documents/upload`
2. 選擇此目錄中的任一檔案
3. 填寫文件 metadata（標題會自動從檔名帶入）
4. 完成上傳流程
